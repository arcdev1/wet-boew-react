  • onAnchorClick(handler: (elem: HTMLAnchorElement) => void): void
    • Defined in routing.ts:13

    Several CDTS components will inject traditional <a> tags (i.e. Anchor Links) for navigation but in a SPA we tend to want to handle routing ourselves. This function will intercept standard anchor links and allow us to provided our own navigation handler.

    export
